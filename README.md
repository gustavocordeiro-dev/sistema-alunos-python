# Sistema de Gerenciamento de Alunos 

Sistema de cadastro e gerenciamento de alunos via terminal, desenvolvido em Python.

## Funcionalidades
- Cadastrar alunos (nome e idade)
- Listar todos os alunos cadastrados
- Buscar aluno por identificação
- Editar dados de um aluno
- Remover aluno da lista

## Tecnologias utilizadas
- Python 3
- Git e GitHub

## Como executar
1. Clone o repositório:
   git clone https://github.com/gustavocordeiro-dev/sistema-alunos-python
2. Execute o arquivo:
   python main.py

## Conceitos aplicados
- Funções e modularização de código
- Listas e dicionários
- Estruturas de repetição e condicionais
- Tratamento de erros com try/except
